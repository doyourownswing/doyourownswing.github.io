# Getting Started with DYOS app

This project was initially bootstrapped with [Create React App](https://github.com/facebook/create-react-app), but was updated to use Vite since CRA is no longer maintained

## Available Scripts

In the project directory, you can run:

### `npm run start`

Runs the app in the development mode.\
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### `npm run deploy`

Deploys the app on github pages, which is live at https://www.doyourownswing.com
